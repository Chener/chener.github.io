响应式设计

当我一年前开始写这个博客时便想写这篇教程。这篇教材适用于所有 jQuery Mobile 开发者，无论你程度如何。我会试着去覆盖从适合多种设备的响应式设计到 C/S 通讯宽泛的重要内容。

介绍

和其他框架不同的是 jQuery Mobile 致力于只支持手机设备。虽然它可以用于桌面端网络开发，但通常结果都会很糟糕的和想象中不一样。当然这不是一定的，这部分教程将告诉你如何成功地在不同设备以及分辨率上使用 jQuery Mobile。首先我们会用简述 CSS media queries 以及 jQuery Mobile，然后会是在大分辨率上的 jQuery Mobile 的使用。

我们将看到

  • 创建一个支持响应式设计的简单图片系统
  • 在页面间传递各种数据
  • 在不同屏幕大小下 jQuery Mobile UI 的使用
