Phonegap 开发

捣鼓了不少时间，原来如此简单。

使用 Phonegap + Ionic 等手机端框架开发接近原生 APP 的方式对熟悉前端开发的程序猿是十分友好的，目前我的开发搭建基本不需要 Eclipse，ADT-bundle 以及 Android Studio 之流，前提是不需要特别复杂的 debug 功能（本来 html 部分就可以通过 Chrome 进行 debug）。

开发阶段

两种方式（目前推荐方式二，安装下就好），均要用到Phonegap Developer App。由于国内 Google play 上不去，国内需登录下载地址

方式一：CLI 开发
  • node 的安装按照步骤一步步来就行，它是一个可供 js 运行的独立环境/引擎，一般 js 都是运行在浏览器提供的引擎上的。
  • CLI: phonegap Vs cordova
  • How the above two differ on adding plugin
  • 关于 config.xml 的位置，现在一般放在 www/ 外。
方式二：使用 Phonegap Desktop App

使用 Desktop App

  • 默认端口 3000，可使用 Desktop App 修改。
  • 默认访问文件 index.html，可在 config.xml 里修改。
  • Phonegap 自带跨域。
  • 无法使用 cookie 进行登录，但 phonegap build 可以！
  • 无法使用第三方插件（待确定）
  • plugins 不光要在 config.xml 里声明， 还需要在 myApp/ 中使用 CLI 输入如 cordova plugin add org.apache.cordova.camera。(待确定->不需要)

部署阶段

Phonegap Build

  • 若没有 config.xml，则 Phonegap Build 目前默认使用 phonegap 3.3.0 （不是最新版本）。
  • 删除 App 的时候如果选择将弹窗禁用后无法删除别的 App 只要将 tab 关闭再重新打开就好。

开发

  • Tutorial: Developing a PhoneGap Application
  • 上面教程完整代码的链接使用 git 下载，或者直接点击网页右下方的下载按钮。下载后只要把 www/ 下的内容替换到使用 Phonegap CLI 或者 Phonegap Desktop App 创建的项目里的 www/。至于 config.xml 使用默认的即可，若需要设置，参考官方文档。
  • Phonegap Events 以及 Plug-in APIs左边目录最下面两个。
  • 按返回键退出
  • [推送]（待确定）
