jQuery Mobile (1.4.5)页面加载与跳转区别：

  • <a> 标签
    可加载同一个 html 页面下的不同 page 或者另一个 html 页面下第一个 page;
  • <a> 标签及 rel = "external"
    可跳转至同一个 html 页面下的不同 page 或者另一个 html 页面下任意 page;
