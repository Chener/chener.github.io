使用 Sublime Text 3

个人认为目前对新老手都是最好的入门编辑器。

知识点

  • Project - Sava Project As ... 就可以把整个项目文件夹保存为两个项目文件 *.sublime-project *.subllime-workspace 以后直接点击这两个文件就可以打开项目了。
  • Console Palette (Shift + Cmd + P) 中可以快速模糊输入各种常用命令。
  • (Cmd + P) 中可以快速切换项目/文件夹中所有文件，如前置输入@则可搜索当前页面的所有方法名。
