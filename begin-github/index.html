Github 入门指南

Github 在这个时代是绕不过去的。

入门常用命令
知识点
  • git checkout branchname 是将本地 workspace 切换为某个本地 branch。
  • git pull origin branchname 更新本地仓库至最新改动。
  • origin 是指 REMOTE_URL。
  • Github 上 repo 的命名习惯使用 - 作为间隔符。
  • to pull from .git repo，需要在 url 后面加上 .git。

使用 SourceTree 代替命令行是初学者不错的选择，比 Github 官方的 GUI 个人感觉要好用。
